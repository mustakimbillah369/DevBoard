Question-1: What are the different ways to select an element in the DOM?

Answer: DOM থেকে এলিমেন্ট সিলেক্ট করার মোট ৫ টি পদ্ধতি রয়েছে। সেগুলো হলোঃ 1. getElementById 2. getElementsByClassName 3. querySelector 4. querySelectorAll 5. getElementsByName

Question-2: What is the difference between innerHTML, innerText, and textContent ?

Answer: innerHTML, innerText ও textContent এর মধ্যে পার্থক্য হলোঃ innerHTML এলিমেন্টের ভিতরের HTML Tag সহ সব কিছু সেট করে। innerText এলিমেন্টের ভিতরে দৃশ্যামান টেক্সট সেট করে। innerContent এলিমেন্টের html tag ও text উভয়ই সেট করে দেখায়।

Question-3: What is event delegation in the DOM?

Answer: Event Delegation হচ্ছে এমন একটা টেকনিক যেখানে আমরা কোনো প্যারেন্ট এলিমেন্ট এ ইভেন্ট বসাই, তারপর সেই ইভেন্ট দিয়ে একাধিক চাইল্ড এলিমেন্ট এর ওপর কাজ করা যায়।

Question-4: What is event bubbling in the DOM?

Answer: Event Bubbling মানে হলো আমরা যখন কোন চাইল্ড এলিমেন্ট এ ক্লিক করি ,তখন সেই ইভেন্ট ধাপে ধাপে উপরের দিকে প্যারেন্ট -> গ্র্যান্ডপ্যারেন্ট -> ডকুমেন্ট পর্যন্ত বাবল করে যায়।

Question-5: How do you create, add, and remove elements using JavaScript?

Answer: ১। নতুন এলিমেন্ট তৈরির জন্য ব্যবহার করতে হবে "document.createElement('tag name')"
